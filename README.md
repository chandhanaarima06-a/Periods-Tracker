# DearMe - Period Tracker App

A personal period tracking app that helps users track their cycle, predict future periods, and get personalized health insights.

## Features (Planned)
- User authentication and personal profiles
- Period cycle tracking and predictions
- Mood tracking
- Food and workout suggestions based on cycle phase
- Personal diary with AI-based advice
- Smart notifications

## Current Status
- Frontend: HTML, CSS, JavaScript (in progress)
- Backend: Coming soon (Spring Boot)
- Database: Coming soon (MySQL/PostgreSQL)

## Tech Stack
- Frontend: HTML, CSS, JavaScript → React
- Backend: Spring Boot (Java)
- Database: MySQL or PostgreSQL

## How to Run
1. Clone this repository
2. Open `frontend/index.html` in your browser
